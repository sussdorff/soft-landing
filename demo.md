# Soft Landing Demo (3-5 min)

## 1. The Problem (30s)

> Snowstorm hits Munich. 8 flights cancelled. 200+ passengers. Today that's hours of manual rebooking. Soft Landing: minutes.

## 2. Architecture (60s)

Show diagram or slides. Hit three points:
- **Lufthansa API** — flight schedules, seat availability, lounges
- **Gemini + Google Search** — hotels, transport, weather, plain-language explanations
- **Real-time WebSocket** — dashboard and mobile app stay in sync

## 3. Live Demo (2-3 min)

Dashboard + mobile app side by side.

1. **Show the disruption** — 150+ passengers visible at once, click a passenger profile
2. **Show options** — HON Circle gets 5-star hotel + lounge; Economy gets 3-star + meal voucher
3. **Passenger submits wish** on mobile — appears instantly in Wish Stream
4. **Airline agent approves** — one click, passenger gets real-time confirmation
5. **Bulk resolve** — agent can also resolve directly from Flight Overview

## 4. Wrap-up (30s)

> 200 passengers, minutes not hours. Real flight data, AI-powered options, real-time sync.
